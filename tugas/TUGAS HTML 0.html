HMTL

HTML adalah singkatan dari Hypertext Markup Language, HTML merupakan salah satu bahasa pengkodean atau pemograman yang digunakan untuk membuat halaman website yang ditampilkan pada web browser. Sebagian besar halaman yang kamu temukan pada internet kebanyakan menggunakan Bahasa HTML.

HTML TAG

Tag HTML adalah alat yang membentuk kata-kata dan kalimat, membantu kita mengkomunikasikan pesan visual ke pengunjung situs. Dari menyusun teks dan gambar sampai menciptakan interaksi yang dinamis, tag HTML bertindak sebagai bahan baku yang mengubah konsep menjadi kenyataan dalam tampilan browser

HTML ELEMENT

Elemen HTML adalah sebuah code html yang sudah disediakan dan setiap code elemen mempunyai kegunaan atau fungsinya masing-masing. Elemen html mempunyai Tag Pembuka dan Tag Penutup, tetapi tidak semua elemen html mempunyai Tag Penutup.

PENGELOLAAN DOKUMENT

Pengelolaan Dokumen adalah proses penerimaan, penyusunan, penyimpanan, pemeliharaan, penggunaan, dan penyajian dokumen secara sistematis.
